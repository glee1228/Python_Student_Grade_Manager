## 프로그램 실행 및 예제

##### 프로그램 실행

```$ python3 project1.py students.txt``` 

Students.txt 를 매개변수 인자로 받아 실행

##### 테이블 검색

```# Show```  

##### 해당 학번 정보 검색

```# search```

##### 해당 학번의 중간, 기말고사 점수 수정

```# changescore```

##### 새로운 학생 레코드 삽입

```# add```

##### 해당 학번 정보 삭제

```# remove```

##### 해당 학점에 해당하는 학생 레코드 검색

```# searchgrade```     

 ##### 프로그램 종료

```# quit```
